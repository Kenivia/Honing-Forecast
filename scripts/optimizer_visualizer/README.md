# Results data explorer

EXTREMELY VIBE CODED thing to help visualize which engine's the best

## Setup

```bash
python -m pip install -r ./scripts/optimizer_visualizer/requirements.txt
```

## Run

```bash
python ./scripts/optimizer_visualizer/app.py
```

Alternatively, run

```bash
npm run optimizer-visualizer
```

The app will open on `http://127.0.0.1:8050/` by default.
