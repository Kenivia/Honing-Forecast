# About

Honing Forecasta is a Lost Ark honing calculator that computes your average gold spent based on your owned mats, and suggests a free tap & juice usage plan that minimizes this average. See more information here: <https://www.reddit.com/r/lostarkgame/comments/1qwskt2/optimize_your_free_tap_juice_usage_with_the_new/>

This project is a work in progress and is VERY messy rn, I do apologize.
